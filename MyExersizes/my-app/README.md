# MyApp

Bu proje, temel Angular uygulama iskeletini ve baslangic yapisini barindirir. Diger calismalar icin referans olarak kullanilir.

## Hedefler

- Proje yapisini tanimak
- Basit sayfa ve component akisini gormek
- Routing ve temel layout altyapisini incelemek

## One Cikan Dosyalar

- `src/app/app.config.ts`
- `src/app/app.routes.ts`
- `src/app/home/`

## Calistirma

```bash
npm install
npm start
```
